# 📊 Netflix Movies & TV Shows — Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project explores how Netflix’s content library has evolved over time using the **Netflix Movies & TV Shows dataset**.  
The goal is to transform raw data into meaningful insights about Netflix’s growth, content strategy, and global expansion.

Through a structured workflow of **data cleaning**, **feature engineering**, and **visualization**, this project uncovers patterns in:

- 📈 Year‑over‑year content growth  
- 🎬 Movies vs TV Shows trends  
- 🌍 Country contributions  
- 🎭 Genre distributions  
- 🔞 Rating categories  
- ⏱️ Duration patterns  

The analysis highlights how Netflix has shifted its focus over the years and provides a data‑driven look at the platform’s evolving catalog.

---

## 🎯 Objectives

This project aims to:

1. **Clean and preprocess** the dataset to handle missing values, inconsistent formats, and mixed data types.  
2. **Engineer new features** (e.g., `year_added`, `duration_int`, `duration_type`) to enable deeper analysis.  
3. **Visualize key trends** that reveal how Netflix’s content strategy has changed over time.  
4. **Tell a clear analytical story** supported by charts, insights, and data‑driven observations.  
5. Produce a **portfolio‑ready notebook** demonstrating practical data science skills.
